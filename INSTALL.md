# Установка на компьютер кинотеатра (Windows)

Программа ставится локально на один ПК. Интернет нужен только для обновлений —
касса работает и без него.

## Что нужно один раз установить на ПК

1. **Node.js** (LTS) — https://nodejs.org → «Next, Next, Finish».
2. **База данных** — один из вариантов:
   - **Docker Desktop** (проще для нас) — https://www.docker.com/products/docker-desktop
     Лаунчер сам поднимет базу.
   - **или PostgreSQL** напрямую — https://www.postgresql.org/download/windows/
     Тогда создайте базу и пропишите доступ в `apps/api/.env` (строка `DATABASE_URL`).
3. **Git** (нужен только для авто-обновлений) — https://git-scm.com/download/win

## Первая установка

1. Скопируйте папку проекта на ПК (или `git clone <адрес репозитория>`).
2. Скопируйте `.env.example` → `.env` и `apps/api/.env.example` → `apps/api/.env`.
3. Двойной клик по **`Запустить-Ailin.bat`**.
   - Первый запуск дольше: ставятся зависимости, собирается программа, создаётся база.
   - Дальше открывается касса в браузере.

Логины по умолчанию: **admin / admin** (администратор), **kassir / cashier** (кассир).
⚠️ Смените пароли и `JWT_SECRET` в `apps/api/.env` перед реальной работой.

## Каждый день

- Просто двойной клик по **`Запустить-Ailin.bat`**.
- Если есть интернет — программа сама проверит обновления в Git, скачает и применит их.
- Если интернета нет — запустится на текущей версии.
- Остановить: закрыть окно лаунчера или **`Остановить-Ailin.bat`**.

Адреса:
- Касса: `http://localhost:3000/`
- Дисплей для гостей: `http://localhost:3000/display`

## Дисплей для гостей (второй экран)

1. Подключите второй монитор/телевизор.
2. Откройте на нём браузер на `http://localhost:3000/display`.
3. Нажмите **F11** — полноэкранный режим (киоск).
   Можно зафиксировать конкретный сеанс: `http://localhost:3000/display?session=<id>`.

Если дисплей — отдельное устройство в той же сети, используйте IP кассы:
`http://<IP-кассы>:3000/display`.

## Авто-запуск при включении ПК (необязательно)

Создайте ярлык на `Запустить-Ailin.bat` и положите его в папку автозагрузки:
нажмите `Win+R` → введите `shell:startup` → Enter → вставьте ярлык.

## Принтер чеков

Установите термопринтер (Xprinter/Epson) как обычный принтер Windows и сделайте его
принтером по умолчанию. При продаже откроется окно печати — выберите формат 80 мм.

## Авто-обновления (как это работает)

Лаунчер при каждом старте, если есть интернет, делает `git pull` из настроенного
репозитория, и при наличии изменений пересобирает программу и применяет миграции БД.
Чтобы это работало, на ПК должен быть склонирован репозиторий с настроенным `origin`
(`git clone <адрес>`), а новые версии — запушены в этот репозиторий.
